How to Run the Online Shopping Portal Using PHP and MySQL

Requirements:

Download and install a local web server such as XAMPP, WAMP, or LAMP.

Start your local web server and ensure Apache and MySQL services are running.

Installation:

Extract the source code zip file.

Copy the extracted folder (e.g., "kyanja") to the web serverâ€™s root directory:

For XAMPP: xampp/htdocs/kyanja

For WAMP: wamp/www/kyanja

For LAMP: var/www/html/kyanja

Locate the provided SQL file ("shopping.sql") inside the extracted folder.

Open PHPMyAdmin in your browser by navigating to http://localhost/phpmyadmin.

Create a new database named shopping.

Import the "shopping.sql" file into the newly created database.

How to Run:

Access the frontend by navigating to http://localhost/kyanja in your browser.

Access the admin panel by navigating to http://localhost/kyanja/admin.
