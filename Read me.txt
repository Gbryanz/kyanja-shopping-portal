Kyanja Shopping Portal  

The Kyanja Shopping Portal is a web-based application built using PHP and MySQL to provide an efficient and user-friendly online shopping experience. It includes separate features for administrators and clients.

## Features  

### Admin Panel  
- Manage Category List  
- Manage Brand List  
- Manage Products  
- Manage Orders / Confirm Orders  
- Insert Products  

### Client Features  
- Browse Products with Details  
- Shopping Cart  
- User Registration and Login  
- Manage Orders  
- Manage Account Details  
- Add Products to Wishlist  
- Track the Progress of Ordered Products  
- Send Messages to Management  

## How to Run the Online Shopping Portal  

### Requirements  
- A local web server like XAMPP, WAMP, or LAMP  
- Apache and MySQL services running  

### Installation  
1. Extract the source code zip file.  
2. Copy the extracted folder (e.g., "kyanja") to your web server's root directory:  
   - For XAMPP: `xampp/htdocs/kyanja`  
   - For WAMP: `wamp/www/kyanja`  
   - For LAMP: `var/www/html/kyanja`  
3. Locate the provided SQL file (`shopping.sql`) inside the extracted folder.  
4. Open PHPMyAdmin in your browser at [http://localhost/phpmyadmin](http://localhost/phpmyadmin).  
5. Create a new database named `shopping`.  
6. Import the `shopping.sql` file into the newly created database.  

### Running the Application  
- **Frontend:** Navigate to [http://localhost/kyanja](http://localhost/kyanja) in your browser.  
- **Admin Panel:** Navigate to [http://localhost/kyanja/admin](http://localhost/kyanja/admin).
